GET / página atual da API
GET /log ?amount=20 Retorna os dados de leitura salvos
GET /lastreading Retorna os dados da última leitura
POST /save Salva os dados de leitura no log do server
PUT /config/ph ?min=7&max=8 Altera os valores de threshold para o PH
PUT /config/temperature ?min=25&max=30 Altera os valores de threshold para a Temperatura
DELETE /clear Limpa os dados de log do server
